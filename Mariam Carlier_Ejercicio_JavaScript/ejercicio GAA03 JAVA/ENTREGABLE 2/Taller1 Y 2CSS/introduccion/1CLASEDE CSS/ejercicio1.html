Clase de css

Hojas de Estilo en Cascada (CSS) es un lenguaje de hojas de estilo que se utiliza para describir la presentación de un documento escrito en HTML o XML (incluidos dialectos XML como SVG, MathML o XHTML). CSS describe cómo deben representarse los elementos en pantalla, papel, voz u otros medios.
